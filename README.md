# cctv-igame-notes

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态形式存储，可直接在浏览器中打开或通过 GitHub Pages 等服务进行托管展示。

## 项目简介

cctv-igame-notes 是一个以 HTML 页面为主要内容的归档仓库。仓库中的页面各自独立，彼此之间没有强依赖关系，适合作为笔记、工具页面或小型演示页面的集合。所有内容均为静态文件，便于快速访问和版本管理。

## 目录说明

```
.
├── index.html          # 入口页面（可选）
├── page1.html          # 示例页面
├── page2.html          # 示例页面
├── assets/             # 资源文件夹（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- `index.html`：仓库的默认入口页面，可根据需要存在或省略。
- 其他 `.html` 文件：每个文件代表一个独立的页面，文件名反映其内容或用途。
- `assets/`：存放页面所需的静态资源，如样式表、脚本、图片等。

## 页面归档说明

每个 HTML 页面都是独立的功能单元，可能包含：

- 信息展示
- 小型工具
- 学习笔记
- 实验性原型

页面之间不共享状态或数据，可直接通过文件路径访问。如需在本地预览，建议使用本地服务器（如 `python -m http.server`）以避免跨域问题。

## 维护说明

- 新增页面：请创建一个新的 `.html` 文件，并确保其路径和资源引用正确。
- 修改页面：可直接编辑对应 HTML 文件，保持代码整洁。
- 删除页面：移除对应文件，并检查是否有其他页面引用。
- 资源管理：请将公共资源放入 `assets/` 文件夹，避免文件分散。

欢迎提交 Pull Request 或 Issue 来改进本仓库。

---

*本仓库仅用于静态 HTML 页面的归档与发布，不涉及任何特定服务或网站。*
